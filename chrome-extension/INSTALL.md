# Quick Installation Guide

## Step 1: Load the Extension

1. Open Chrome
2. Go to `chrome://extensions/`
3. Turn on **Developer mode** (top-right toggle)
4. Click **Load unpacked**
5. Navigate to and select the `chrome-extension` folder
6. The extension should appear in your list

## Step 2: Verify Installation

1. Look for the Glass extension icon in Chrome's toolbar
2. Click it to open the popup
3. It should show connection status

## Step 3: Use It

1. Start your Glass Electron app (`npm start`)
2. Open any webpage in Chrome
3. Click the Glass extension icon
4. Click "Read Current Tab"
5. Go back to Glass and ask questions about the page!

That's it! No command-line flags needed.
